🏥 乳腺癌良恶性诊断分类
SVM 与 BP 神经网络对比实验

📋 实验概览

  • 数据集：Breast Cancer Wisconsin (Diagnostic) —— 共 569 例样本（良性 357，恶性 212）
  • 特征数：30 个数值特征（细胞核的半径、纹理、周长、面积、光滑度、紧密度、凹度等）
  • 问题背景：医学辅助诊断。通过对细针抽吸活检（FNA）图像的数字化特征进行机器学习，辅助医生判断肿块是良性还是恶性。
  • 实验目的：比较 SVM（支持向量机）与 BP 神经网络在同一分类任务上的性能差异，并分析原因。
  • 软件环境：PyCharm + Python 3.13 + scikit-learn 1.8 + matplotlib + seaborn

🔍 结果分析与讨论

1. 准确率 / F1 / AUC 总体表现

两个模型在该数据集上均表现优秀，AUC 都达到了 0.9969，说明模型的整体排序能力极强。BP 神经网络在准确率和 F1 上略微领先（0.9720 vs 0.9580），但两者差距很小（约 1.4 个百分点）。这说明对于这一"线性可分性较好"的数据集，线性 SVM（kernel=linear）已经能胜任，而神经网络通过增加非线性进一步提升了少量边界样本的识别精度。

2. 精确率 vs 召回率的权衡

SVM 的召回率（0.9889）非常高（把良性样本几乎全找对），但精确率偏低（0.9468）——意味着它把一些恶性样本误判为良性（5 例假阴性）。在医学诊断场景中，假阴性（漏诊恶性）是更危险的错误。

BP 神经网络的精确率与召回率都达到 0.9778，两个类别的表现更均衡。假阳性 2 例，假阴性 2 例，总体误差更少（4 vs 6）。

3. 计算效率对比

在训练阶段（含网格搜索），两者耗时接近（5s vs 4.1s）。但在预测阶段，BP 神经网络的预测速度是 SVM 的 3 倍以上（0.29ms vs 0.96ms）。原因：SVM 需要计算测试样本与所有支持向量的距离，而 MLP 只需一次前向传播。对于大数据集或实时预测，这一差异会被放大。

4. 最佳超参数的启示

  • SVM 选择了 kernel=linear, C=0.01：小的 C 值意味着更强的正则化，表明数据集本身噪声不大，不需要复杂核函数。
  • BP-NN 选择了 (64, 32) 双层 + tanh 激活 + alpha=0.0001 + lr=0.01：两层网络已经足够，更深的网络并没有带来明显收益，体现了"小数据用浅模型"的经验法则。

5. 结论

在乳腺癌良恶性分类这一任务上，SVM 和 BP 神经网络都是非常优秀的方案，二者的 AUC（0.9969）完全持平，BP 神经网络的综合指标略优。

  • 推荐 BP 神经网络：若追求准确率和预测速度，且数据特征之间存在非线性关系时，优先选择。
  • 推荐 SVM：当样本量较小、需要强的可解释性（线性核的权重可直接解释特征重要性）、或需要更少的超参数调优时，SVM 更稳定和简单。

实际医学应用中，不应只看准确率，还应结合领域知识与医生需求，对模型的决策阈值做进一步的调整（例如降低判为良性的阈值来减少漏诊）。
